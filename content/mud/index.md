# PUNNARAJ Mutual Understanding Document

This repository is the living **MUD — Mutual Understanding Document** for the PUNNARAJ project. It preserves project intent, decisions, architecture, and working memory in Markdown so humans and AI workers can collaborate without losing context.

- [[30-workflows/human-workspace|Human Workspace]] — create, update, upload, delete, and publish flow.

## Operating principle

Time is the irrecoverable resource. The repo should make project memory fast to capture, fast to publish, and hard to lose.
