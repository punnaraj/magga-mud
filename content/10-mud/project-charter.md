# Project Charter

PUNNARAJ uses this repository as a durable shared memory space. The core artifact is a network of Markdown notes that can be read as plain files, edited in Obsidian, reviewed in Git, and published as a website.

## Purpose

1. Preserve mutual understanding between humans and AI workers.
2. Make decisions and assumptions explicit before they become hidden debt.
3. Publish selected knowledge as a navigable web page.
4. Keep the source portable: Markdown first, Git controlled, static-site deployable.

## Roles

- **Human steward**: decides what is canonical, private, or public.
- **AI worker**: drafts, links, summarizes, and proposes updates quickly.
- **Repository**: acts as the memory substrate and publication source.
- **Static site**: exposes the public and reviewable MUD.

## Canonical workflow

Capture first, refine second. Every significant conversation should create or update a note, decision, or task in this vault.
